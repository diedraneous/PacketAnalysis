---- Introduction ----

Packet analysis is a fundamental skill in cybersecurity, enabling analysts to closely examine network traffic to understand typical behavior and identify anomalies. To build and strengthen this skill, these labs and packet captures provide hands-on exercises that help beginners develop foundational knowledge while allowing more experienced learners to reinforce and expand their practical expertise.

To get started, download the "Packet Analysis Lab" document. In there, you will find instructions, as well as questions to help you work through the packet captures and think more about how the process of packet analysis works. 

Once you are ready to analyze the packets, download them from the main page.

---- What you will need ---

1. Wireshark (https://www.wireshark.org)
2. Word or Document Editor that can open Word Files
3. A want to learn and explore packet analysis
